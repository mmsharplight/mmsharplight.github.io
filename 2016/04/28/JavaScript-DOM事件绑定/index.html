JavaScript DOM事件绑定
tabindex属性：用来当前HTML元素节点是否被tab键遍历，以及遍历的优先级。

  • 如果 tabindex = -1 ，tab键跳过当前元素。
  • 如果 tabindex = 0 ，表示tab键将遍历当前元素。如果一个元素没有设置tabindex，默认值就是0。

  • 如果 tabindex 大于0，表示tab键优先遍历。值越大，就表示优先级越大。
